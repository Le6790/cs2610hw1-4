This page lists and describes usful git commands.

This page was constructed with information from https://www.html-5-tutorial.com/, https://developer.mozilla.org/en-US/docs/Web/HTML/Element, https://www.siteground.com/tutorials/git/commands.htm, and part of Professor Falor's Git Cheat Sheet class example.

Table1

Name Description Example
git init Initializes a new git repository git init
git add filename ... Adds a set of changes to the repository git add index.html
git commit Saves a snapshot of your work, along with your explanation git commit
git status Shows you the status of files in the index vs working directory. Lists changed files, files not currently in git directory git status
git diff Display the differences between committed code and the working files git diff
git remove removes a file from your index and working directory git remove
git remote add Connect a remote repository to this one. git remote add origin https://LeKent@bitbucket.org/LeKent/cs2610.git
git push Send changes to a remote repository git push origin master
git clone Makes a git repository copy froma remote source. git clone https://LeKent@bitbucket.org/erikfalor/weatheranalyzer.git
git grep Lets you search through your tree of contents for words and phrases git grep index.html
